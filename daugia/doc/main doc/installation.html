Installation Process

Prerequisites PHP

  • Minimum PHP Version 7.2.5

Installation

Auctioneer is a shared hosting supported application. to install follow steps below

  • Step 1 : Extract The project file to your public directory of your hosting server.

  • Step 2 : Go to YourDomain.com. you will see this and follow the instruction.

  • Step 02: This step will say if your server has met all requirements.

  • Step 03: This step will say if All directory Permission is properly available. You must have permission to the following directory and files.

  • resource/lang

  • bootstrap/cache

  • public/uploads/

  • Step 04: This step contains the app name, url and installation environment

  • Step 05: This step contains all database information

  • Step 06: This step contains all email smtp information

  • Step 07: Filling up this step with the initial user data allows you to get your very first admin user

That's all. there will be a success message and installation has been done.
